# ม ทวีภัณฑ์ (M.T.T.) — เว็บไซต์ร้านค้า

เว็บไซต์ร้านอุปกรณ์ไฟฟ้า & ฮาร์ดแวร์ **ม ทวีภัณฑ์** เป็น static site (HTML/CSS/JS ล้วน)
พร้อม deploy ขึ้น Vercel ได้ทันที ออกแบบให้ **ต่อยอดเพิ่มสินค้า/หมวดหมู่ได้ง่าย**

## โครงสร้าง

```
index.html                    หน้าแรก — แบรนด์ฮับ (หมวดหมู่ + สินค้าเด่น + ติดต่อ)
products/
  jet-lighter.html            หน้าขาย "ไฟฟู่ M.T.T." (ตะกร้า + QR พร้อมเพย์ + สั่งผ่าน LINE)
assets/
  css/theme.css               ดีไซน์กลาง (โทนสี/ปุ่ม/การ์ด/หัว-ท้าย) ใช้ร่วมทุกหน้า
  js/shop-config.js           ⭐ ตั้งค่าร้านที่เดียว: PromptPay, LINE, เบอร์โทร, ที่อยู่
  js/site.js                  เมนูมือถือ + สลับภาษา TH/EN + เติมข้อมูลติดต่อ
  js/vendor/qrcode.min.js     ไลบรารีสร้าง QR (ฝังในโปรเจกต์ ไม่พึ่ง CDN)
vercel.json                   ตั้งค่า clean URL
```

## ตั้งค่าก่อนใช้จริง

แก้ไฟล์เดียว: **`assets/js/shop-config.js`**

| ค่า | คือ |
|-----|-----|
| `PROMPTPAY_ID` | เบอร์พร้อมเพย์ หรือเลขบัตรประชาชนของร้าน (ใช้สร้าง QR) |
| `LINE_URL` / `LINE_ID` | ลิงก์ & ไอดี LINE OA |
| `PHONE`, `EMAIL`, `ADDRESS_TH`, `HOURS_TH` | ข้อมูลติดต่อ (แสดงทั้งเว็บอัตโนมัติ) |

ราคา/แพ็กเกจของไฟฟู่แก้ได้ในตัวแปร `PRODUCTS` ท้ายไฟล์ `products/jet-lighter.html`

## เพิ่มสินค้า/หมวดหมู่ใหม่ (ต่อยอด)

- **หมวดหมู่หน้าแรก** → แก้ array `CATEGORIES` ใน `index.html`
  (ตั้ง `live:true` + ใส่ `href` ไปหน้าสินค้าเมื่อพร้อมขาย)
- **สินค้าเด่นหน้าแรก** → แก้ array `FEATURED` ใน `index.html`
- **หน้าสินค้าใหม่** → ก็อป `products/jet-lighter.html` เป็นเทมเพลต แล้วปรับข้อมูลสินค้า
  (ทุกหน้าใช้ `theme.css` + `shop-config.js` ร่วมกัน จึงหน้าตาเป็นชุดเดียวกันอัตโนมัติ)

## Deploy

อัปโหลดทั้งโฟลเดอร์ขึ้น Vercel (Add New → Project) — ไม่ต้อง build
หน้าไฟฟู่จะอยู่ที่ `/<โดเมน>/products/jet-lighter`

## ฟีเจอร์

- ดีไซน์ industrial-retail (หมึกกรมท่า + เหลืองอำพัน) รองรับมือถือเต็มรูปแบบ
- สลับภาษา ไทย/อังกฤษ (จำค่าไว้ให้)
- ตะกร้าสินค้า + สร้าง QR พร้อมเพย์ยอดตรงอัตโนมัติ (มาตรฐาน EMVCo)
- ปุ่มคัดลอกออเดอร์ & เปิด LINE, ปุ่ม LINE ลอยทุกหน้า
- SEO/Open Graph meta ครบ
